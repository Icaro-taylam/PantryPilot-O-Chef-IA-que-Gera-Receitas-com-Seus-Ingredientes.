# PantryPilot-O-Chef-IA-que-Gera-Receitas-com-Seus-Ingredientes.
PantryPilot: Seu Chef IA Pessoal. Você informa os ingredientes que tem na cozinha, e a inteligência artificial do Gemini (do Google) cria na hora uma receita original, deliciosa e que usa tudo! É a solução perfeita, fácil de usar no celular, para evitar desperdício.
